# React Color Picker App

A simple **Color Picker Application** built using **React.js** that allows users to select and preview colors dynamically.
The app demonstrates React fundamentals such as **state management, event handling, and dynamic UI updates**.

---

## 🚀 Live Demo

https://colorpickerforyou.netlify.app/

---

## ✨ Features

* Real-time background color preview
* Displays selected color value (HEX)
* Simple and responsive UI
* Built with React functional components

---

## 🛠 Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Vite
* Material UI
